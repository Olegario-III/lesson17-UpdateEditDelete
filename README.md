# Edit & Delete Users React App

## Overview
This React app demonstrates **CRUD operations** (Create, Update, Delete) on a user list. Users can:
- **Add** new users
- **Edit** existing users
- **Delete** users
- Toggle between **Add** and **Update** mode using a single form

Perfect for **Lesson 17: Edit and Delete Users**

---

## Features
- Add new users with name and email
- Edit any user (form auto-fills with current data)
- Delete users instantly
- Dynamic button text: **"Add User"** or **"Update User"**
- Clean, responsive UI

---

## Technologies Used
- **React** + **useState**
- **JavaScript (ES6)**
- **CSS** (via `App.css`)
- No external dependencies
